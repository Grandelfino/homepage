## 🚀 Quick start

1. **開発環境の構築**

    Docker composeを用いてローカルに開発環境を構築します．

    ```shell
    git clone https://github.com/Grandelfino/homepage
    ```

2. **Docker環境の実行**

    ローカル環境でのサイト立ち上げ方法

    ```shell
    cd homepage/
    docker compose up -d
    ```

3. **表示のテスト**

    ブラウザで[localhost:8000](http://localhost:8000)にアクセスする
